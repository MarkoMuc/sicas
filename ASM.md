# Assembler

- `.` are comments.

- Numbers:
    - Immeditate operand is denoted with `#`
    - BINARY:
        - B'NUMBERS' -> BIN
        - 0bNUMBERS -> BIN
    - HEX:
        - X'numbers'-> HEX
        - 0xNUMBERS -> HEX
    - FLOATS:
        - F'' -> FNUM
        - 0fNUMBERS-> FNUM
    - Decial numbers -> NUMS
